## Production notes

**Captions:** lime (`#a8d946`) on black for max contrast. Inter or SF Pro Bold, min 32pt. Hold each caption ≥ 1.5s. Hand-typed — auto-generated isn't tight enough.

**Music:** Tycho-style ambient, ducked to ~30% under voiceover. 60s version gets a small drop at the 0:36 montage. Use Epidemic Sound, Artlist, or YouTube Audio Library — never copyrighted tracks.

**Recording the Claude Desktop clips:** record once, reuse in both demo + promo. Light theme reads better against the cream/lime palette.

**Where to post:**
- **X:** version A pinned to profile during judging window
- **LinkedIn:** version A or B (B if builder/VC audience)
- **YouTube Shorts:** version B
- **DM to `contact@sns.id`:** Loom of B with a short personal note
- **Frontier hackathon Discord / Superteam discord:** version A as teaser
- **Reddit r/solana:** version A *only* with substantive post body — raw video posts get downvoted

**File naming:**
```
pitch/promo-30s-vertical.mp4

pitch/promo-claude-scene.mp4   ← reusable MCP clip
```
